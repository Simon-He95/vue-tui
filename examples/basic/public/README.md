# Video demo asset

`video-demo.mp4` is a six-second excerpt from 08:00–08:06 of
[Big Buck Bunny 60fps 4K](https://www.youtube.com/watch?v=aqz-KE-bpKQ), the official Blender
Foundation upload.

Big Buck Bunny © 2008 Blender Foundation / www.bigbuckbunny.org, licensed under
[CC BY 3.0](https://creativecommons.org/licenses/by/3.0/). The excerpt was cropped in time,
downscaled to 640×360, reduced to 12fps, transcoded to H.264, and stripped of audio for the
vue-tui terminal video demo.
