# @simon_he/vue-vite-conf-card

Vue x Vite conference terminal card CLI powered by vue-tui.

## Usage

Recommended:

```bash
npx @simon_he/vue-vite-conf-card
```

You can also install it first and run the CLI command directly:

```bash
npm i -g @simon_he/vue-vite-conf-card
vue-vite-conf-card
```

## Notes

- The executable command name is `vue-vite-conf-card`.
- Prefer `npx @simon_he/vue-vite-conf-card` over `npx -p @simon_he/vue-vite-conf-card vue-vite-conf-card`, since the `-p` form may fail to expose the bin command in some npm environments.
